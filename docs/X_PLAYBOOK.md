# T4 — X (Twitter) Execution Playbook

**Project:** Zotopie  
**Role:** Growth Engineer  
**Status:** Operational

---

## Overview

X is the distribution engine for software comparison content. The format advantage: compare pages and review data translate naturally into high-engagement threads. The audience — founders, marketers, indie hackers — actively follows software discovery content.

**Goal:** 1,000 followers + 300 referral visits/month within 90 days.

## 3. Compare Page → X Thread

This is the core content engine. Every compare page becomes one thread.

**Thread structure (6–8 tweets):**

```
Tweet 1 (Hook):
[Tool A] vs [Tool B] — which should you use in 2025?

I compared both in depth. Here's the breakdown 🧵

---

Tweet 2 (TL;DR):
TL;DR:

→ Choose [Tool A] if: [use case 1]
→ Choose [Tool B] if: [use case 2]
→ Neither if: [edge case]

---

Tweet 3 (Pricing):
💰 Pricing

[Tool A]: Free plan → $X/mo (paid)
[Tool B]: Free trial → $Y/mo (paid)

[1–2 sentence insight about value/pricing]

---

Tweet 4 (Key Feature 1):
⚡ [Key differentiator — e.g., "Ease of use"]

[Tool A]: [2 sentences]
[Tool B]: [2 sentences]

Winner: [Tool X] for most users

---

Tweet 5 (Key Feature 2):
🔌 [Key differentiator — e.g., "Integrations"]

[Tool A]: [X]+ integrations including [example]
[Tool B]: [Y]+ integrations, better for [use case]

---

Tweet 6 (Verdict):
🏆 My take:

If you're [user type 1]: [Tool A]
If you're [user type 2]: [Tool B]
If you're bootstrapped: [budget option]

---

Tweet 7 (CTA):
Full side-by-side with feature table, pricing breakdowns, and who each is best for:

👉 zotopie.com/compare/[pair]

(No affiliate deal with either — editorial only)

---

Tweet 8 (Engagement):
What tool are YOU using for [use case]? 

Drop it below 👇 I read every reply.
```

**UTM for thread link:** `?utm_source=x&utm_medium=thread&utm_campaign=compare-[pair]`


## 9. Growth Mechanics

### Follow Strategy

Follow 50 accounts/day from these categories:
- Founders building in public (>1K followers, tweet about tools)
- Marketing professionals (job titles: CMO, Head of Marketing, etc.)
- Indie hackers posting about their stack
- SaaS product accounts in Zotopie categories

**Ratio target:** Never exceed 1.1 following:followers ratio.

### Reply Strategy (30 min/day)

Search for these on X daily:
```
"best [category] tool"
"[tool name] alternative"
"switching from [tool name]"
"[tool A] vs [tool B]"
"help me choose"
```

Reply with genuine insight + link only if directly relevant and adds value beyond a search.

### Quote Tweet Strategy

When software accounts post product updates:
```
Quote the announcement + "Here's how [Tool] compares to alternatives in [category]: [link to alternatives page]"
```
